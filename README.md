# Route Optimizer v6 — Streamlit App

## Запуск

```bash
pip install -r requirements.txt
streamlit run route_optimizer_app.py
```

## Вхідний файл (.xlsx)

Файл має містити 3 листи:

| Лист | Опис | Обов'язкові колонки |
|------|------|---------------------|
| **sh1** | Точки відвідування | `track`, `FullName`, `Latitude`, `Longitude`, `zero_Latitude`, `zero_Longitude` |
| **sh2** | Дати виїздів | `DateActivity`, `track` |
| **sh3** | Плановий пробіг | `Трек`, `Період`, `План, км` |

## Параметри (бокова панель)

| Параметр | За замовч. | Опис |
|----------|-----------|------|
| TOLERANCE | 0.10 | ±10% від плану |
| MAX_STOPS | 5 | Макс. точок/день |
| N_SAMPLES | 3000 | Розмір бібліотеки Monte Carlo |
| SEED | 42 | Зерно випадковості |
| MAX_ITER | 8000 | Ітерації fine-tune |
| COVERAGE_BONUS | 3.0 | Бонус за нові точки |
| REPEAT_PENALTY | 0.3 | Штраф за повтори |
| MAX_REST_RATIO | 0.6 | Ліміт днів без виїзду |

## Як працює

1. Завантажте файл → перегляньте дані та карту
2. Якщо є пропущені координати — визначте їх кліком на карті
3. Налаштуйте параметри в боковій панелі
4. Натисніть «Запустити оптимізацію»
5. Програма:
   - Завантажує відстані через OSRM Table API (кешує в SQLite)
   - Будує бібліотеку маршрутів (Monte Carlo)
   - Розподіляє маршрути по дням (scored fill + fine-tune)
6. Завантажте результат `v4_optimized.xlsx`
